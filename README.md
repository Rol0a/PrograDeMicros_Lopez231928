# Programación de Microcontroladores — IE2023

Este repositorio está destinado al uso de **iteración, validación y calificación** de los laboratorios desarrollados para el curso **Programación de Microcontroladores [IE2023]**, correspondiente al **Ciclo 1 de 2026**, impartido por **Pedro Castillo**.

El repositorio incluye la creación y desarrollo de **laboratorios, proyectos y tareas** relacionadas con la programación del **Arduino Nano**, basado en el microcontrolador **ATMega328P**.

Los contenidos aquí presentados forman parte del proceso de aprendizaje práctico del curso y reflejan la aplicación de conceptos fundamentales de programación y control de microcontroladores.
